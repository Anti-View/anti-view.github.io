说些什么之一 #

这篇文章写于2021年年底，曾经在朋友圈也发过，后来放到了公众号里，但这些媒介都差点意思，想了想还是写到个站里了。

八周的设计实践课结束了。说是八周，从九月到十二月，也足足过了三个月；说是结束，论文还没个截止日期，所以也算不上真的结束；说是“设计实践”，其实也没有实践，因为我们班的课题没有企业合作，是个空头课程。

我不得不把这些话写在这里，因为沉默不全是处事的金针。在此，我无意争论其他班或者老师是否比我们的更好一些——这种争论没有意义。我只是看到了太多充满求知欲与探索欲的学生，把自己的希望与信任地交给某些课程，或者说某些老师。当然，结果烂透了。

一个很显然的结论是，对于一个做梦都想争一流大学的艺术院校，管好艺术项目，做好政治任务，拿更多国际大奖才是当务之急。对老师或教授来说，拉到足够的经费发表足够的论文，满足学院要求才是生存大计。相比之下，本科生教学显得无利可图（尤其在刚刚评上博士学位授权点的情况下）。要说传道授业解惑，也肯定要先把奶喂给自己工作室的硕士博士，而不是大一二三四里的某个本科生。就算是项目任务不太重的一些任课教师，或者没评上硕导的教师，他们也不会为这么多人负责——“学不懂？那是因为你没别人用功。为什么跟你一个班上的某某某同学，人家就能懂？”教学评估会考察出勤率是否达到指标，有几个人迟到，但绝不会考察班上到底有几个人在老师的言传身教中学到了东西，或者这些老师根本就没有言传身教过。当然，真正在课堂上言传身教的教学，其质量是不会有人过问的。

而对于具体课程，教学内容的陈旧程度也令人叹为观止。当然，以“教学经验”的名义，十年用同一本教科书是可以理解的。但倘若是教科书也没有呢？从大一到大三，从未见过哪位老师正儿八经地用过什么教科书上课，提都没提过。某位老师更甚，开设一门叫什么“产品xx学”的课，公然在课堂上称：“我不教你们产品xx学，你们自己去网上搜网课听。”没有权威的影印版教科书暂且不说，外文作者的中文译版也从未见人用过，充其量不过是所谓国内“专家”的专著，但是，你只要瞪圆了眼睛一看就会发现——那上面署的“专家“名字竟然就是他自己。

一份PPT，从2008年用到2021年；再或者PPT也没有，直接拿自己当年写的腐臭论文来上课，反反复复讲数十年。大一的也好，大四的也罢，或者去其它学校开的讲座，无一例外都是在讲同一套内容，也难怪同一碗冷饭炒了又炒，根本没人愿意吃。谁能容忍对于一门只有几十年历史的新兴学科，在我们的教学内容中，最新的知识仅涉及到上个世纪八十年代的“新技术”？这样的课程，之所以能够存在并延续下去，从根本上讲是因人设课——开设相应课程就是为了不让部分教师下岗。这也无怪国外有华裔学者实在看不下去了，拍案疾呼：“中国大学，怎么那么盛产「活化石」！”

这八周的课里，有位老师说的话尤其有趣，我现在都还会时不时想起来反复咀嚼：“做学生就得忍，忍你就能学到知识。我们当学生的时候，老师还直接撕我们的方案呢。现在的学生，一点委屈都忍不了，所以学不到东西。”

是啊，忍，学生是该学会忍的，不止学生，任何人都该学会忍。但一面也要那位老师确实有些真材实料在身上，这才能够安心地忍，合乎情理地忍，忍得开心，忍得快乐。学生挖空心思准备，第二天还没来得及汇报一个字，就被某位八周只见过两次面的“教授”堵上了嘴，剩下的时间就是在听那位“教授”满嘴放烟花似地说着些什么“人”“品”“域”的，你再追问，他又摆出一副沉默的高深莫测的臭脸孔。这些词，本来是无论哪一个人都能懂的，无非是用户、产品、环境这些词汇，但你要知道，他们是有个叫做“权威”的面具戴在脸上的，谁若是当众把它撕下来，他又要满嘴放烟花，拿生僻字出来跟你说道了。

挖空心思汇报方案的学生们，与同样挖空心思看方案的老师们，构成了一个完美的圆环。在二者日复一日的机械劳动中，我只看到纸张、电力以及粮食不断被浪费，却看不到知识的领域在拓展，看不到学术的漏洞被填补。我无意否定这份可贵的精神，但这份精神充其量只能称为悲壮。耗费了大量的时间和精力做出来的那些“方案”，根本登不上真正的知识殿堂。真正能成为上课理由的，只有对真正的知识的渴望。旅程图画得再熟练，模型做得再美丽，ppt页数堆得再多，知识的大厦有因此而多出一块砖么？说真的，“人”“品”“域”也好，“忍”也罢，都是大厦旁边恶臭的大粪，尽管让这些大粪笼罩在那些“学识不凡”的头脑上吧。
